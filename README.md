# YTIG
youtube id grabber, TBD

# Installation
1. Create a folder:
```
youtube-link-saver
```
2. Put the 3 files inside it.
3. Open Firefox.
4. Go to:
```
about:debugging
```
5. Click:
```
This Firefox
```
6. Click:
```
Load Temporary Add-on
```
7. Select:
```
manifest.json
```

# Usage
1. Open any webpage.
2. Right click anywhere.
3. Click:
```
Copy visible YouTube links
```
4. The extension copies all visible links on the page and puts it in your clipboard.
