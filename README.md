# YTIG
youtube id grabber, TBD

# Installation
1. Open Firefox.
2. Go to:
```
about:debugging
```
3. Click:
```
This Firefox
```
4. Click:
```
Load Temporary Add-on
```
5. Select:
```
manifest.json
```

# Usage
1. Open any webpage.
2. Right click anywhere.
3. Click:
```
Copy visible YouTube links
```
4. The extension copies all visible links on the page and puts it in your clipboard.
